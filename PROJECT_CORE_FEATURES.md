# Soulart / Ideart 项目核心功能说明（完整版本）

> 文档时间：2026-03-10  
> 说明范围：基于私有完整版项目代码梳理的核心功能点清单（不含敏感实现细节与密钥）

## 1. 项目定位

Soulart（Ideart）是一个面向设计与内容生产的 AI 工作平台，核心目标是把「生成、编辑、协作、发布、运营」放进同一套工作流。产品同时覆盖：

- C 端创作者的 AI 画布创作与素材管理
- 模型驱动的图像/视频/3D/音频生成与后处理
- 多租户商户化运营（套餐、支付、内容配置、后台权限）
- 管理后台的运营看板、配置中心与审计体系

## 2. 功能域总览

- 用户与账号体系
- AI 画布编辑器（含图层系统与多模态编辑）
- 智能聊天与流式任务执行
- AI 模型路由与多供应商编排
- 图像/视频/3D/音频生成与高级编辑 API
- 项目、文件与素材资产体系
- 灵感社区与案例回放
- 会员积分与支付结算
- 多租户商户与平台运营能力
- Admin V2 后台与 RBAC 权限系统
- 安全防护与密钥治理体系
- 国际化与站点配置系统

## 3. 用户与账号体系

### 3.1 登录注册能力

- 支持手机号验证码登录（OTP）
- 支持账号密码登录（手机号/邮箱 + 密码）
- 支持第三方登录（Google、WeChat）
- 支持首次注册自动建档与邀请关系绑定
- 支持邀请奖励积分发放（邀请人 + 被邀请人）

### 3.2 会话与身份信息

- 基于 NextAuth 统一会话管理
- Session/JWT 中扩展了业务字段：
- `user.id`
- `role`
- `merchantId`
- `points`
- `hasPassword`
- `isAdmin`

### 3.3 账号安全

- 密码错误次数限制（Redis 计数 + 窗口期）
- 验证码 TTL 与限流策略
- 账号敏感操作附带安全策略可配置化（运行时策略）

## 4. AI 画布编辑器（核心工作台）

### 4.1 画布与图层系统

- 图层类型覆盖：
- `rect`、`circle`、`triangle`、`star`
- `text`、`path`
- `image`、`video`
- `gen_frame`（生成帧）
- `video_gen_frame`（视频生成帧）
- `smart_board`
- `group`（分组）

- 图层属性支持：
- 坐标/尺寸/缩放/旋转/透明度/混合模式
- 填充（纯色、渐变、图案、图片）
- 描边、圆角、字体、对齐、字重、装饰
- 图像调整参数（曝光、对比、饱和度、锐化、颗粒等）
- 图层特效（阴影、外发光、描边、内阴影、浮雕等）

### 4.2 交互与工具栏

- 选择与手型模式
- 形状绘制工具（矩形、线条、箭头、圆、多边形、星形）
- 文本与路径工具
- 铅笔/钢笔工具
- 上传入口（图片、视频）
- 智能画板创建
- 视频生成帧创建
- 工具栏图标与位置可配置化

### 4.3 画布高级能力

- 图层历史快照与版本回溯
- 选区浮层与上下文操作条
- 画布标记点（Marker）与局部分析联动
- 生成结果回填到图层（含上传态/失败态）
- 图层组合与父子层级管理

### 4.4 画布外观可运营化

- 画布背景主题配置（多背景样式）
- 画布表面颜色体系配置
- 商户级个性化配置隔离

## 5. 智能聊天与流式任务执行

### 5.1 聊天能力

- 侧边栏聊天输入与多模态附件
- 支持对话历史、消息结构化存储
- 支持代理（Agent）角色切换与模式切换

### 5.2 流式执行（SSE）

- 客户端消费 SSE 流数据
- 任务分阶段状态展示（思考/生成/完成/错误）
- 支持增量文本拼接（delta）与步骤化回放

### 5.3 生成控制参数

- 生成模式：图像 / 视频
- 比例、分辨率、画质、时长、方法、模式
- 多镜头视频提示词（multi-shot）
- 首尾帧、参考图、参考视频注入
- 自动模型选择与手动选型并存

## 6. AI 模型路由与多供应商编排

### 6.1 模型层

- 统一模型类型定义（image/video/3d/audio/chat）
- 模型参数标准化（比例、分辨率、时长、批量数、模式等）
- 模型运行时 ID（含 provider + model）解析

### 6.2 路由层

- `ModelRouter` 统一调度多供应商
- 支持官方模型与 Relay 模型混合路由
- 支持按供应商能力回退（fallback）
- 支持 DB 动态供应商配置优先于静态环境配置

### 6.3 供应商覆盖

- OpenAI / OpenAI-compatible
- Google Gemini
- Volcengine / Doubao / Jimeng
- Kling
- Alibaba/Qwen
- Replicate
- Tripo3D
- Vidu

### 6.4 可靠性机制

- 请求队列并发控制
- 指数退避重试（含 429/5xx/网络类错误识别）
- 供应商故障自动切换与降级

## 7. 生成与编辑 API（创作能力矩阵）

### 7.1 生成主链路

- `POST /api/generate`
- 图像/视频/3D/音频任务统一入口
- Prompt 解析增强（比例、数量、分辨率等）
- 并发策略、限流、积分扣减、任务编排

### 7.2 图像编辑能力

- `POST /api/edit-image`：多图编辑、模型自适配
- `POST /api/edit-text`：图文编辑（文本替换/修改）
- `POST /api/remove-bg`：抠图去背景
- `POST /api/erase`：擦除/修复（含 mask）
- `POST /api/outpaint`：外扩图（扩图）
- `POST /api/auto-mask`：视觉定位自动选区
- `POST /api/vectorize`：位图转矢量
- `POST /api/explode`：图像分层拆解
- `POST /api/upscale`：智能放大（含多档分辨率）

### 7.3 视觉理解与检测

- `POST /api/ocr`：OCR 文字识别
- `POST /api/qr-detect`：二维码定位检测

### 7.4 视频与音频能力

- `POST /api/video-to-audio`：视频转音效/配乐
- `GET /api/video-to-audio`：任务状态轮询
- `POST /api/mockup/*`：Mockup 贴图与表面识别

## 8. 项目、文件与资产体系

### 8.1 项目管理

- 项目创建、读取、更新、列表
- 支持轻量视图（cards/recent）优化返回体
- 用户项目隔离与权限校验

### 8.2 文件管理

- 生成文件（图片/视频/3D/音频）归档
- 文件元数据：尺寸、时长、缩略图、模型、Prompt
- 文件删除与归属校验

### 8.3 上传体系

- `POST /api/upload` 支持文件上传
- 上传限流与文件大小限制
- 本地 URL 与 OSS URL 映射

### 8.4 个人素材资产

- 资产分类（按媒体类型）
- 资产项（文件、缩略图、MIME、大小）
- 用户侧素材沉淀与复用

## 9. 灵感社区与案例回放

### 9.1 灵感内容体系

- 分类管理（category）
- 案例管理（case）
- 步骤回放（step）
- 用户交互（点赞/收藏）

### 9.2 前台体验

- 灵感流列表与分类筛选
- 案例详情页
- 回放页支持逐步消息播放与图片切换

### 9.3 数据结构

- `inspiration_categories`
- `inspiration_cases`
- `inspiration_case_steps`
- `inspiration_case_user_actions`

## 10. 会员积分与支付体系

### 10.1 会员能力

- 套餐定义（价格、积分、并发限制、权益）
- 用户会员状态刷新与到期管理
- 按套餐并发控制与优先级调度

### 10.2 积分能力

- 模型调用扣点
- 每日赠送与邀请奖励
- 扣点失败拦截（不足或模型受限）

### 10.3 订单与支付

- 订单创建、查询、状态流转
- 支持月付/年付
- 支持支付方式：
- 支付宝（密钥模式/证书模式）
- 微信支付
- 线下转账

### 10.4 支付配置中心

- 商户级支付参数管理
- 支付配置校验（证书/密钥完整性）
- 支付设置变更审计

## 11. 多租户商户化系统

### 11.1 商户模型

- 商户基础信息、状态、联系人、品牌配置
- 用户绑定商户隔离
- 商户级计划、模型、供应商、订单、项目隔离

### 11.2 商户入驻申请

- 商家申请提交与审核流
- 供应商 API 信息提交（加密存储）
- 审核结果与备注回传

### 11.3 平台/商户双视角

- 平台管理员可跨商户管理
- 商户管理员仅能管理自身商户数据

## 12. Admin V2 后台系统

### 12.1 看板与运营

- 经营概览（用户、会员、销量、销售额）
- 趋势图与套餐占比
- 实时队列状态与等待时延指标
- 最近订单列表

### 12.2 后台管理域

- 用户管理、用户角色管理
- 商户管理、商户申请管理
- 套餐管理、订单管理、结算统计
- 模型管理、供应商管理、参数模板管理
- 支付设置、站点设置、首页配置、法务内容配置
- 灵感内容管理、新闻管理
- 安全策略管理（运行时）

### 12.3 权限体系

- RBAC：角色、菜单、角色菜单、用户角色
- 平台级与商户级数据域控制
- 管理操作审计日志

## 13. 安全与治理

### 13.1 网关与访问控制

- 代理中间层统一鉴权
- 公共路径白名单
- Admin 区域角色校验
- API 级限流（通用/AI/Auth 分桶）

### 13.2 SSRF 防护

- 图片/视频代理域名白名单
- 私网 IP 检测与 DNS 解析防护
- 只允许可信上游域访问

### 13.3 敏感信息治理

- 供应商密钥 AES-GCM 加密存储
- 敏感字段脱敏展示
- 内部请求签名校验（HMAC + 时间窗）

### 13.4 运行时安全策略

- 密码重试上限
- 短信频率与 TTL
- API 流量阈值
- 策略支持后台动态下发

## 14. 国际化与站点配置

### 14.1 多语言

- 支持 10 种语言
- 本地语言偏好存储
- 翻译键类型安全

### 14.2 站点可视化配置

- 首页文案、导航、分类、特性区块、公告弹窗
- 品牌信息与 Logo
- Footer 信息（公司、社媒、备案、版权）
- 法务页面（条款/隐私/客服）动态配置

## 15. 数据模型核心实体（摘要）

- 用户域：`User`、`Session`、`Account`、`VerificationCode`
- 商户域：`Merchant`、`MerchantApplication`
- 创作域：`Project`、`Message`、`GeneratedFile`
- 资产域：`AssetCategory`、`AssetItem`
- 模型域：`ModelProvider`、`ModelConfig`
- 会员域：`MembershipPlan`、`Order`、`PointsLog`、`MembershipContent`、`PaymentSettings`
- 管理域：`AdminRole`、`AdminMenu`、`RoleMenu`、`UserRole`、`AdminAuditLog`

## 16. 技术栈与架构实现（摘要）

- 前端：Next.js App Router + React + TypeScript
- 状态管理：Zustand
- 数据层：Prisma + SQLite（并有 Redis 辅助）
- 异步与队列：Redis + 内存回退队列
- 对象存储：OSS（文件上传与资源托管）
- 认证：NextAuth
- 运营后台：Ant Design + ECharts

## 17. 开源版与私有版边界说明

本次公开仓库为“安全精简版”，仅保留基础前端结构与静态页面。以下能力不在开源版中提供：

- 完整画布引擎与编辑实现
- 完整聊天/Agent 编排链路
- 完整后台管理系统
- 大模型路由与供应商调用实现
- 数据库/Redis/云厂商集成实现

如需在开源版中扩展，请基于公开代码独立实现，不要直接迁移私有生产逻辑。
